# Archipelago Core Client for Minecraft

A minimal client between Minecraft and one (or more) Archipelago multiworld slots.

| Repository                                                                      | Description                                    |
|---------------------------------------------------------------------------------|------------------------------------------------|
| Core Client                                                                     | Minimal Minecraft-to-Archipelago client mod    |
| [Vanilla Randomizer](https://github.com/ArchipelagoMinecraft/VanillaRandomizer) | Vanilla randomizer implementation mod          |
| [KubeJS Archipelago](https://github.com/ArchipelagoMinecraft/KubeJSArchipelago) | KubeJS integration for Core Client             |
| [Vanilla APWorld](https://github.com/ArchipelagoMinecraft/VanillaAPWorld)       | Archipelago APWorld for the vanilla randomizer |

## KubeJS Integration

See the [KubeJS Archipelago](https://github.com/ArchipelagoMinecraft/KubeJSArchipelago) repository.

## Maven Dependency

Requirements: 
1.7.10 : unimixins-all https://github.com/LegacyModdingMC/UniMixins
1.8.9 : mixinbooter https://github.com/CleanroomMC/MixinBooter/
1.12.2 : mixinbooter https://github.com/CleanroomMC/MixinBooter/


Also https://github.com/eigenraven/MinecraftDev is nice to have for <=1.12.2 IDE support

Important: run the "Reset active project" gradle task before committing!
